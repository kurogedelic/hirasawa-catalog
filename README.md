# [hirasawa-catalog](https://kurogedelic.github.io/hirasawa-catalog/)

平沢進氏およびその周辺プロジェクト（P-MODEL, 核P-MODEL等）の全楽曲を研究するため、
楽曲を網羅し、厳格な作品番号を付加し管理するオープンソース・データベースプロジェクトです。

## データの修正・提案
データの誤り、過不足などをみつけましたら、issueを開いて教えていただけるとありがたいです。
